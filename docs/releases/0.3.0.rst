0.3.0
----------

Breaking Changes
~~~~~~~~~~~~~~~~

- N/A

New Features
~~~~~~~~~~~~

- The extract_data module now produces either the old .ext files (w/o PIO) or a fully-formed child boundary file (w/ PIO).
- Ability to specify `reference_date` in the namelist. (`#300 <https://github.com/CWorthy-ocean/ucla-roms/pull/300>`_)

Bug Fixes
~~~~~~~~~


- Initialize `forcing_version` string to prevent errors from CDR forcing module (`#303 <https://github.com/CWorthy-ocean/ucla-roms/pull/303>`_)
- Bugfixes from initial PIO testing (`#306 <https://github.com/CWorthy-ocean/ucla-roms/pull/306>`_)
- Tests were missing were the new restoring force functionality. (`#299 <https://github.com/CWorthy-ocean/ucla-roms/pull/299>`_)
- Fix a PIO deadlock related to skipping reads (`#307 <https://github.com/CWorthy-ocean/ucla-roms/pull/307>`_)
- Guard against other deadlock conditions (`#307 <https://github.com/CWorthy-ocean/ucla-roms/pull/307>`_)
- Fix a bug where rivers could not be read in with PIO activated (`#309 <https://github.com/CWorthy-ocean/ucla-roms/pull/309>`_)
- Restore ability to use passive tracers, which was accidentally removed

Improvements
~~~~~~~~~~~~


- Adds exact restart test (`#301 <https://github.com/CWorthy-ocean/ucla-roms/pull/301>`_)
- Adds test that default cppkey set has not changed (`#301 <https://github.com/CWorthy-ocean/ucla-roms/pull/301>`_)

Miscellaneous
~~~~~~~~~~~~~

- Adds github CI action to rebuild CI testing containers, either on-demand (via a workflow dispatch action) or whenever one of the containerfiles changes. (`#303 <https://github.com/CWorthy-ocean/ucla-roms/pull/303>`_)
- Partial implementation of CDR Tracer (not fully functional yet, still in development)
